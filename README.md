## 🛠 آموزش تصویری نصب افزونه وردپرس

1. فایل `gglct.php` را در مسیر `wp-content/plugins` بارگذاری کنید.
2. از پیشخوان وردپرس، فعال‌سازی را انجام دهید.
3. منوی **بهینه‌ساز GGLCT** به صفحه مدیریت افزوده می‌شود.
4. تنظیمات را با چند کلیک مدیریت و شخصی‌سازی کنید.
